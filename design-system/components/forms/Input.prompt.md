Labelled text field — use for all single-line entry (search, forms, filters).

```jsx
<Input label="Tracking code" placeholder="DK-0000-0000-BR" leftIcon={<SearchIcon/>} hint="Find any package" />
```

Props: `label`, `leftIcon`, `hint`, `error` (turns border red + shows message), `size` (`sm|md|lg`), `disabled`. Focus shows an orange ring.
