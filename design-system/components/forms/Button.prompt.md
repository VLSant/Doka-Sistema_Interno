Doka's primary action control — use for any commit/submit/CTA; reach for `ghost`/`outline` for secondary actions.

```jsx
<Button variant="primary" size="md" leftIcon={<TruckIcon/>}>Track shipment</Button>
```

Variants: `primary` (orange), `secondary` (plum), `outline`, `ghost`, `danger`. Sizes `sm|md|lg`. Props: `loading`, `disabled`, `fullWidth`, `leftIcon`, `rightIcon`. Primary lifts with an orange glow on hover; all variants shrink slightly on press.
