# Tracking Portal — UI Kit

The **customer-facing** package-tracking page: a friendlier, marketing-adjacent surface
with the brand hero, a tracking search, and a live shipment journey.

## Files
- `index.html` — renders the portal.
- `TrackingPortal.jsx` — nav, hero search, and the tracking `Result` (journey stepper, timeline, map, driver, details).

## Primitives used
Button, Input, Badge, StatusPill, ProgressTracker, Avatar, Card.

## Notes / liberties
Representative recreation (no original Doka source was provided). Plum header band +
orange hero gradient follow the brand's two-color personality. Map is a placeholder.
